# 🚀 Getting Started

## Prerequisites

Before running the project, make sure you have:

* Android Studio Hedgehog or later
* JDK 11 or higher
* Android SDK 24+
* Kotlin Support Enabled

---

# 📥 Installation

## 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/scientific-calculator-app.git
```

## 2️⃣ Open in Android Studio

```
cd scientific-calculator-app
```

* Open Android Studio
* Click on **Open**
* Select the project folder

---

## 3️⃣ Sync Gradle

Wait for Gradle sync to complete automatically.

---

## 4️⃣ Run the App

* Connect your Android device
  OR
* Start an Android Emulator

Then click:

```
Run ▶
```

OR press:

```
Shift + F10
```

# 🧠 Expression Evaluation Logic

The app uses a **custom-built mathematical expression evaluator** that follows proper:

* BODMAS Rules
* Bracket Priority
* Operator Precedence
* Scientific Function Parsing

This avoids dependency on third-party calculator engines and improves learning of internal logic implementation.
